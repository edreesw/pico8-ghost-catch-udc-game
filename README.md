# pico8-ghost-catch-udc-game

#Underground Death Collector / Ghost Catch

Welcome to the Underworld Death Collections Department. Your job is to catch souls coming down the waterfall to get them ready for processing. Watch out for the red ones though, those nasty little guys are above your paygrade so just leave them to someone more experienced downstream. Good luck. 

##Controls

Move left and right using the arrow keys on your keyboard. 

---
##Info

Game was created using the Fantasy Console, Pico-8.
